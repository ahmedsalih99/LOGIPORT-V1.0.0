## ✨ المميزات الرئيسية

- **إدارة المعاملات** — استيراد / تصدير / ترانزيت مع ترقيم ذكي تلقائي لا يتكرر
- **تتبع الكونتينر** — مراحل الشحن البحري مع Timeline مرئي تفاعلي
- **إدارة العملاء والشركات** — ملفات كاملة مع جهات الاتصال والبنوك والشركاء
- **توليد المستندات الجمركية** — 9 أنواع من الفواتير والمستندات بصيغة PDF
- **نظام تسعير متكامل** — أسعار حسب شروط التسليم (Incoterms) والمسار
- **لوحة تحكم تحليلية** — إحصائيات فورية وآخر المعاملات والمهام العاجلة
- **دعم متعدد اللغات** — عربي وإنجليزي وتركي مع تبديل فوري دون إعادة تشغيل
- **نظام صلاحيات ديناميكي** — أدوار ومستويات وصول قابلة للتخصيص بالكامل
- **وضع مظلم / فاتح** — ثيم احترافي مع تبديل فوري
- **سجل تدقيق شامل** — تتبع كل عملية على مستوى النظام
- **تحديثات تلقائية** — يتحقق من GitHub Releases عند كل تشغيل
- **مزامنة سحابية** — دعم Supabase اختياري للعمل متعدد الأجهزة

## 🚀 التثبيت والتشغيل

### للمستخدم النهائي
شغّل ملف `LOGIPORT_Setup_1.0.0.exe` واتبع خطوات معالج التثبيت.

### للمطورين (من المصدر)

```bash
# 1. انسخ المستودع
git clone https://github.com/ahmedsalih99/LOGIPORT-V1.0.0.git
cd LOGIPORT-V1.0.0

# 2. أنشئ بيئة افتراضية (مُوصى به)
python -m venv venv
venv\Scripts\activate

# 3. ثبّت المكتبات
pip install -r requirements.txt

# 4. شغّل التطبيق
python main.py
```

### أول تشغيل
عند الفتح لأول مرة يظهر **معالج الإعداد الأولي** (Setup Wizard) لإنشاء حساب المدير وضبط الإعدادات الأساسية.

> **⚠️ مهم:** احتفظ ببيانات حساب Admin في مكان آمن.
